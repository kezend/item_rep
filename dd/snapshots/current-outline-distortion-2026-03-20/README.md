# Typography Scanner Engine

Stable web-based 3D typography tool built with Three.js for editorial and fashion visuals.

## Features

- Minimal step-1 scene that always renders something visible
- Shared typography geometry rendered as mesh, point cloud, or outline
- Glow toggle, color, and intensity
- Bend, twist, wave, and noise distortion
- Single object mode and multi-variation grid mode
- Randomized layout via rows, columns, and seed
- Camera presets: front, perspective, top, editorial
- Typography controls: text, font, upload, size, tracking, line height
- Export: PNG, JPG, GIF, SVG, GLTF, OBJ
